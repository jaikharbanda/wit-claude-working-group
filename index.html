Your free skill: anti-AI writing style

Makes Claude (or any AI) write like a human — it strips banned words, kills the "it's not X, it's Y" tell, and removes the AI giveaways.

How to use it
  • Claude desktop / Cowork: download the .skill file and click "Save skill". Claude then applies it whenever it writes.
  • Browser Claude / ChatGPT / Gemini: download the .md file and paste it into your Project or custom instructions.
